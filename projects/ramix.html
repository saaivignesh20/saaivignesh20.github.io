Ramix: Cloud ESL Tags Platform

Full-Stack Development

Technology Stack

Ramix is an advanced solution for managing Electronic Shelf Tags (ESL), offering a seamless and intuitive interface for efficient retail operations. The software solution enables users to manage merchants, stores, and products, link or unlink tags, and add or delete gateways effortlessly.

The platform also features a customizable ESL tag templating system, a robust Role-Based Access Control (RBAC) mechanism for secure feature access, a logging system to monitor in-store operations, and a dynamic dashboard that provides tailored statistics based on user roles. The authenticated access is powered by Auth0, ensuring that the application is protected using latest industry standards.

Ramix is supported on all major platforms all major web browsers. It is also a progressive web application (PWA) that can be installed seamlessly. The highlighted feature is that it encompasses a custom-developed ZXing-based QR/barcode scanner that helps manage devices, products and gateways easily when using handheld devices such as a tablet or a mobile phone.

Solution Offered:
  • UI/UX Design of the proposed solution done using Figma design tool, with continuous feedback and improvements.
  • Full-Stack Development with a mono-repo repository architecture
  • Deployment and Monitoring on Amazon Web Services
